# CommunityConnect

[![Watch the demo](https://img.youtube.com/vi/VUhvHzAOk6Y/maxresdefault.jpg)](https://youtu.be/VUhvHzAOk6Y)

CommunityConnect is a community-focused application built with React / React Native (Expo) that enables users to connect, engage, and interact through a clean, modular, and scalable front-end architecture.



## Overview

This project focuses on building a modern, component-driven application using React patterns such as reusable components, context-based state management, and page-based navigation.



## Demo

Click the image above to view a short demo of the application.

## Features

* Modular, reusable UI components
* Context API for shared/global state
* Page-based navigation structure
* Mobile-first, responsive design


## Tech Stack

* React / React Native
* Expo
* JavaScript
* Context API



## Installation

### Prerequisites

* Node.js
* npm or yarn

### Setup

Clone the repository:

```
git clone https://github.com/ShikharGupta77/CommunityConnectAWProject.git
cd CommunityConnectAWProject
```

Install dependencies:

```
npm install
```

Start the development server:

```
npm start
```

This will start the Expo development server.



## Running the App

* Install **Expo Go** on your iOS or Android device
* Scan the QR code shown after running `npm start`
* The app will load on your device
